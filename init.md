We will be creating a new app. This app will be used to showcase comic pages. We will have a dashboard in which the user will be able to create a showcase proyect. In it, he will be able to upload the comics page and assets. 

When editing a project, the user will be able to add/remove pages and assets. He will also be able to edit the page and asset properties.

The showcase will be paginate, and each page can be either a single assets, or a framing of a page. The user will be able to set up each step/page and it will be displayed in the order he sets up. 

I want also a feature that will allow the user to add sounds and sound effects and music to the showcase. Each step will allow the user to start or stop the sound or sound effect. Also, the user will be able to trigger single sound effects by pressing numerical buttons.

Showcase will have a preview mode, in which the user can preview the showcase and see how it will look like.



Stack

- Next JS (latest)
- Tailwind CSS (latest)
- Shadcn UI (latest)
- Lucid (latest)
